不定尺寸图片居中的一些记录

Jan 23rd, 2015 9:26 pm | Comments

这是在实习工作中遇到的一个需求，实现未知尺寸的图片在一固定宽高的容器中水平和垂直居中。

比如是在一个200px的正方形容器，图片的宽高可能大于200px,也可能小于200px。对于小图片，就是简单的水平和垂直居中，而大图片则要求按比例缩放，并实现居中。

因为图片大小未知，且可能超出容器，很多垂直居中的方法不能用。

解决方法最终如下：

HTML大概是这样的：

<div class="box">
  <img src="img/0.jpg">
</div>

CSS：

.box {
  width: 200px;
  height: 200px;
  position: relative;
  text-align: center;
}

.box2 img {
  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
  margin: auto;
  max-width: 200px;
  max-height: 200px;
}
